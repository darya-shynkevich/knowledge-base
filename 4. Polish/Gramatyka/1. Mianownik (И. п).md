***Прилагательные*** => ==***-e** / **-ie==**  : piękne kobiety , drogie rowery, wysokie tytuły*

**МУЖСКОЙ** (**MĘSKOOSOBOWY**)

*DWAJ / TRZEJ / CZTEREJ*

	1. chłopiec => chłopcy
	2. inżynier => inżynierzy
	3. partner => partnerzy
	4. mechanik => mechanicy
